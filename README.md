## Тематика сайта, название

Игра «Танчики на двоих». Идея сделать игру, где два игрока будут управлять танками, пытаясь победить друг друга, при этом им будут мешать монстры, из которых выпадают улучшения для танка.


## Целевая аудитория

Друзья, которые хотят поиграть друг против друга, имея при этом только один компьютер. Их потребность — весело провести время.

## Функциональные возможности

* настройка силы и количества монстров;
* выбор карты(поля) для игры;
* настройка цвета поля и танков;
* настройка количества жизней игроков;
* статистика побед;
* генерация бонусов при уничтожении монстров
